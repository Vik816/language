# Language Practice Bot

Telegram-бот для разговорной практики языка с разбором ошибок через Claude API.

## Структура проекта

```
language-bot/
├── package.json
├── .env.example      ← шаблон ключей, скопировать в .env
├── .env              ← твои реальные ключи (создать самому, не публиковать!)
├── data/
│   └── users.json    ← создаётся автоматически, хранит прогресс пользователей
└── src/
    ├── index.js        ← точка входа
    ├── bot.js          ← логика команд и диалога
    ├── claudeClient.js ← общение с Claude API
    ├── storage.js      ← хранение данных пользователей
    └── scenarios.js    ← список тем и уровней
```

## Шаги запуска

1. Установи Node.js (версия 18 или новее) с сайта nodejs.org, если ещё нет.
2. Создай бота в Telegram: напиши @BotFather → команда `/newbot` → следуй инструкциям → скопируй токен.
3. Получи API-ключи на console.proxyapi.ru (или console.anthropic.com, если есть доступ напрямую):
   - ключ для Anthropic (диалог и разбор ошибок)
   - ключ для OpenAI (распознавание речи Whisper + синтез речи TTS)
4. Установи ffmpeg — он нужен, чтобы превращать синтезированную речь в формат голосовых сообщений Telegram:
   - Windows: скачать с ffmpeg.org, добавить в PATH (или `choco install ffmpeg`, если есть Chocolatey)
   - Mac: `brew install ffmpeg`
   - Linux: `sudo apt install ffmpeg`
   Проверить, что установился: `ffmpeg -version`
5. В папке проекта выполни:
   ```
   npm install
   ```
6. Скопируй `.env.example` в `.env` и впиши туда токен бота, ключ Anthropic и ключ OpenAI.
7. Запусти бота:
   ```
   npm start
   ```
8. Открой своего бота в Telegram и напиши `/start`. Можешь писать текстом или отправлять голосовые — бот ответит соответственно.

## Команды бота

- `/start` — начать, выбрать язык
- `/level` — сменить уровень сложности
- `/scenario` — сменить тему диалога
- `/stats` — посмотреть свой прогресс
